# 실전 연습 문제 - 과외 방식에 따른 성적 차이 검정 (Independent Samples T-test Exercise)

이 문서는 집단 간 평균 점수를 비교하는 **독립표본 t-검정(Independent Samples t-test)** 실전 연습 문제 및 모범 답안 예시입니다.

---

## 1. 연구 시나리오 및 데이터셋 정의

### 📋 시나리오
교육 연구소에서 새로운 온라인 자기주도형 학습 과외 시스템(실험군)과 기존 오프라인 과외 시스템(대조군)의 학업 성취도 효과를 비교하려고 합니다. 총 20명의 학생을 무작위로 절반씩 배정하여 4주간 학습을 진행한 후 기말고사 성적을 측정했습니다.

### 📊 데이터셋 구성
* **독립변수 (`group`)**: 과외 방식 (범주형)
  * `1` = 실험군 (온라인 자기주도형 학습)
  * `2` = 대조군 (기존 오프라인 학습)
* **종속변수 (`score`)**: 기말고사 성적 (연속형, 0~100점 척도)

| 학생 ID | 집단 변수 (`group`) | 기말고사 성적 (`score`) |
| :--- | :--- | :--- |
| S01 ~ S10 | 1 (실험군) | 85, 88, 79, 92, 81, 84, 87, 90, 78, 83 |
| S11 ~ S20 | 2 (대조군) | 76, 78, 82, 70, 74, 80, 75, 79, 71, 75 |

---

## 2. 실전 문제

> **[문제]** 
> 새로운 온라인 자기주도형 학습 과외 시스템(실험군)을 적용한 학생들과 기존 오프라인 학습(대조군)을 적용한 학생들 간 기말고사 성적(`score`) 평균에 유의한 차이가 있는지 유의수준 5%($\alpha = .05$)에서 검정하시오.

---

## 3. 분석 단계별 가이드 및 모범 해설

### 1단계: SPSS 변수 보기 설정
1. `group` 변수: 유형은 `숫자`, 측정은 `명목(Nominal)`으로 지정한 후 **값(Values)** 항목에서 `1`에 `실험군`, `2`에 `대조군` 레이블을 할당합니다.
2. `score` 변수: 유형은 `숫자`, 측정은 `척도(Scale)`로 지정합니다.

### 2단계: 분석 실행 경로
* **[분석] ➔ [평균 비교 및 비례] ➔ [독립표본 t-검정]**
* **검정 변수**: `score`
* **집단 변수**: `group` (집단 정의 클릭 후 집단 1에 `1`, 집단 2에 `2` 입력)

### 3단계: 가상의 SPSS 출력 결과
분석 실행 시 나타나는 결과 테이블의 핵심 통계치는 다음과 같습니다.

#### 집단 통계량 표
* **실험군**: N = 10, 평균 = 83.70, 표준편차 = 4.47
* **대조군**: N = 10, 평균 = 76.00, 표준편차 = 3.94

#### 독립표본 검정 표
* **Levene의 등분산 검정**: 유의확률(Sig.) = .691 ($\ge .05$ 이므로 등분산 가정이 충족됨)
* **'등분산이 가정됨' 행의 통계치**:
  * $t$값 = 4.085
  * 자유도 ($df$) = 18
  * 유의확률 양측 ($p$) = .001 ($< .05$)

---

## 4. 최종 논문 보고서 서술 예시 (APA 양식)

독립표본 t-검정을 통해 새로운 온라인 자기주도형 학습(실험군)과 기존 오프라인 학습(대조군) 간 기말고사 성적 평균 차이를 분석한 결과는 다음과 같다. 

분석에 앞서 Levene의 등분산 검정을 실시한 결과, 두 집단의 분산은 동질적인 것으로 나타났다($F = 0.163, p = .691$). 따라서 등분산 가정을 충족하는 $t$ 통계량 값을 기준으로 분석을 진행하였다.

분석 결과, 온라인 자기주도형 학습을 수행한 실험군의 성적 평균($M = 83.70, SD = 4.47$)은 기존 오프라인 학습을 수행한 대조군의 성적 평균($M = 76.00, SD = 3.94$)보다 통계적으로 유의하게 높은 것으로 검정되었다($t(18) = 4.09, p = .001$). 즉, 새로운 온라인 자기주도형 학습 과외 시스템이 기존 방식보다 성적 향상에 더 유의미한 효과가 있음을 알 수 있다.
