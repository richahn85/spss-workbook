# 실전 연습 문제 - 부서에 따른 직무 만족도 차이 검정 (One-way ANOVA Exercise)

이 문서는 3개 이상의 집단 간 평균값 차이를 검정하는 **일원배치 분산분석(One-way ANOVA)** 및 **Scheffe 사후분석** 실습을 위한 실전 연습 문제 및 모범 답안 예시입니다.

---

## 1. 연구 시나리오 및 데이터셋 정의

### 📋 시나리오
어느 기업의 인사담당자가 근무 부서 유형(영업부, 연구소, 인사과)에 따라 조직원들의 직무 만족도에 차이가 존재하는지 분석하고자 합니다. 총 15명의 직원을 무작위 추출하여 소속 부서 정보와 직무 만족도를 수집했습니다.

### 📊 데이터셋 구성
* **독립변수 (`dept`)**: 근무 부서 (범주형)
  * `1` = 영업부 (Sales)
  * `2` = 연구소 (R&D)
  * `3` = 인사과 (HR)
* **종속변수 (`satisfaction`)**: 직무 만족도 점수 (연속형, 1~10점 척도, 10점일수록 높음)

| 직원 ID | 부서 변수 (`dept`) | 직무 만족도 (`satisfaction`) |
| :--- | :--- | :--- |
| S01 ~ S05 | 1 (영업부) | 6, 7, 5, 6, 6 (평균 = 6.00) |
| S06 ~ S10 | 2 (연구소) | 8, 9, 8, 9, 8 (평균 = 8.40) |
| S11 ~ S15 | 3 (인사과) | 5, 4, 6, 5, 5 (평균 = 5.00) |

---

## 2. 실전 문제

> **[문제]** 
> 부서별 직무 만족도(`satisfaction`)의 평균 차이가 통계적으로 유의미한지 유의수준 5%($\alpha = .05$)에서 분산분석을 실시하고, 차이가 있다면 Scheffe 검정을 사용해 구체적으로 어느 부서 간 차이가 발생하는지 검정하시오.

---

## 3. 분석 단계별 가이드 및 모범 해설

### 1단계: SPSS 변수 보기 설정
* `dept` 변수: 유형 `숫자`, 측정 `명목(Nominal)`. 값 ➔ `1` = `영업부`, `2` = `연구소`, `3` = `인사과`
* `satisfaction` 변수: 유형 `숫자`, 측정 `척도(Scale)`

### 2단계: 분석 경로 및 옵션 설정
* **경로**: **[분석] ➔ [평균 비교 및 비례] ➔ [일원배치 분산분석]**
* **종속변수**: `satisfaction`, **요인**: `dept`
* **사후분석 클릭**: `Scheffe` 체크
* **옵션 클릭**: `기술통계`, `분산 동질성 검정` 체크

### 3단계: 가상의 SPSS 출력 결과

#### 1) 분산 동질성 검정 (Levene의 등분산 검정)
* 유의확률(Sig.) = .691 ($\ge .05$ 이므로 세 집단의 분산이 동질적이라는 가정이 충족됨)

#### 2) 분산분석 (ANOVA) 표
* **집단간 제곱합 (Between Groups)** = 31.600, 자유도 ($df$) = 2, 평균제곱 = 15.800
* **집단내 제곱합 (Within Groups)** = 6.800, 자유도 ($df$) = 12, 평균제곱 = 0.567
* **$F$값** = 27.882
* **유의확률 (Sig.)** = .000 ($p < .05$) ➔ 부서별 직무 만족도 간 유의미한 차이가 존재함.

#### 3) Scheffe 사후분석 표 (다중 비교)
* **영업부(6.00) vs 연구소(8.40)**: 평균 차이 = -2.400, 유의확률(Sig.) = .001 ($p < .05$) ➔ 차이 유의함.
* **영업부(6.00) vs 인사과(5.00)**: 평균 차이 = 1.000, 유의확률(Sig.) = .177 ($p \ge .05$) ➔ 차이 유의하지 않음.
* **연구소(8.40) vs 인사과(5.00)**: 평균 차이 = 3.400, 유의확률(Sig.) = .000 ($p < .05$) ➔ 차이 유의함.

---

## 4. 최종 논문 보고서 서술 예시 (APA 양식)

부서에 따른 직원들의 직무 만족도 점수 평균에 차이가 있는지 분석하기 위해 일원배치 분산분석(One-way ANOVA)을 실시하였다. 

분석을 수행하기에 앞서 Levene의 분산 동질성 검정을 실시한 결과, 집단 간 분산은 동질적인 것으로 나타났다($F = 0.384, p = .689$). 

분석 결과, 소속 부서 유형에 따른 직원들의 직무 만족도 점수 평균은 통계적으로 유의미한 차이가 있는 것으로 나타났다($F(2, 12) = 27.88, p < .001$). 

구체적으로 어떤 부서 간에 만족도 차이가 발생하는지 판별하기 위해 Scheffe의 사후비교 검정을 실시하였다. 분석 결과, 연구소 직원의 직무 만족도 평균($M = 8.40, SD = 0.55$)은 영업부 직원($M = 6.00, SD = 0.71$)과 인사과 직원($M = 5.00, SD = 0.71$)의 만족도 평균보다 통계적으로 유의하게 높은 수준을 보였다. 반면, 영업부 직원과 인사과 직원 간의 직무 만족도 평균 차이는 통계적으로 유의하지 않은 것으로 검정되었다. (사후검정 결과: 연구소 > 영업부 = 인사과)
