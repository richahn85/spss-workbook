# SPSS AMOS PROCESS macro 매개효과 조절효과 분석

이 PDF는 양적 논문 작성자가 매개효과, 조절효과, 조절된 매개효과를 실제 논문 사례 중심으로 따라 할 수 있게 만든 분석 가이드다. 머리말은 각 사례를 `통계 프로그램 사용법 -> 분석 결과물 -> 통계표 -> 해석` 흐름으로 제시한다고 설명한다.

## 구조

- Part 1은 SPSS, AMOS, PROCESS macro 사용법을 소개한다.
- Part 2는 SPSS, AMOS, PROCESS macro를 활용한 매개효과 분석을 다룬다.
- 이후 조절효과와 조절된 매개효과는 PROCESS macro 중심으로 확장된다.

## 인제스트 포인트

이 자료는 [[spss-regression|회귀분석]]에서 한 단계 나아가 변수 간 인과 경로를 모델링하는 법을 다룬다. PROCESS macro는 회귀 기반 모형을 반복 실행하고 부트스트랩 신뢰구간으로 간접효과를 평가하는 실무 도구로 정리할 수 있다. ^[inferred]
