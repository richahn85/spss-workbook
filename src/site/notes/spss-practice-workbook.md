# SPSS 실전 문제집 프로젝트

이 프로젝트는 `_sources/SPSS 실전 문제집` 폴더의 PDF 5개와 `SPSS30 Data` 실습 데이터를 Obsidian에서 학습 가능한 wiki로 엮는다. 원문은 보존하고, wiki에는 분석법 선택, 실행 절차, 출력 해석, 데이터 카탈로그를 연결한다.

## 핵심 참고자료

- [[spss-practice-nje-workbook-2024]] - SPSS 화면을 따라가는 N제 문제집
- [[excel-spss-r-statistics-practice-problems]] - 통계학 입문 문제은행
- [[spss-30-manual]] - SPSS 30 장별 매뉴얼
- [[spss-amos-process-mediation-moderation]] - 매개/조절/PROCESS macro 분석
- [[kaggle-data-strategy-practice-spss-r-python]] - Kaggle 데이터 전략 실무 연습
- [[spss30-data-catalog]] - 장별 실습 데이터 파일 카탈로그

## 학습 로드맵

1. [[spss-data-types]]로 변수의 측정수준과 역할을 잡는다.
2. [[spss-output-interpretation-workflow]]로 SPSS 결과표 읽는 순서를 익힌다.
3. [[spss-running-t-test]], [[spss-running-anova]], [[spss-running-regression]]으로 핵심 추론 분석을 반복한다.
4. [[mediation-moderation-and-conditional-process]]와 [[process-macro-analysis-workflow]]로 논문형 경로분석 주제로 확장한다.
5. [[kaggle-data-strategy-practice-spss-r-python]]를 통해 예측 모델과 전략 도출 문제로 확장한다.
