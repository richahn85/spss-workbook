# SPSS R Python Kaggle 데이터 전략 실무 연습

이 PDF는 코딩 자체보다 데이터가 주어졌을 때 문제를 정의하고, 분석 절차를 설계하고, 결과를 전략으로 연결하는 경험을 강조한다. 머리말은 R/Python 코드도 제공하지만 본문은 데이터 분석 경험이 적은 독자를 위해 SPSS 중심으로 실습한다고 설명한다.

## 목차 기반 범위

- Kaggle 플랫폼 탐색과 데이터 분석 프로세스
- 고객 이탈, 직원 퇴직, 개인 신용 위험 등 이진 분류 예측
- 고객생애가치와 부동산 가격 같은 연속변수 예측
- 상권, 지하철 탑승객, 대기질 등 현황 분석
- 의료보험과 신용카드 부정거래 같은 이상탐지

## Wiki에서의 역할

이 자료는 [[spss-regression|회귀분석]]과 로지스틱 회귀, 의사결정나무, 신경망 모델을 SPSS 실습 커리큘럼의 후반부로 확장한다. 특히 분석 결과를 "전략 도출"로 연결한다는 점에서 [[spss-practice-curriculum-map|커리큘럼 지도]]의 응용 단계에 배치된다. ^[inferred]
